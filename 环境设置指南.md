## 📱 验证连接

设置完成后，验证 ADB 是否正常工作：

```bash
# 检查 ADB 版本
adb version

# 查看已连接的设备
adb devices
```

如果看到设备列表，说明连接成功！

## 🔧 手动设置（可选）

如果你想手动操作：

### 1. 解压文件
```powershell
Expand-Archive -Path "C:\Users\18268\android-sdk\platform-tools.zip" -DestinationPath "C:\Users\18268\android-sdk" -Force
```

### 2. 设置环境变量
```powershell
# 设置 ANDROID_HOME
[Environment]::SetEnvironmentVariable("ANDROID_HOME", "C:\Users\18268\android-sdk", "User")

# 添加到 PATH
$path = [Environment]::GetEnvironmentVariable("PATH", "User")
[Environment]::SetEnvironmentVariable("PATH", "$path;C:\Users\18268\android-sdk\platform-tools", "User")
```

### 3. 验证
```bash
# 重启命令行后运行
adb version
```

## ⚠️ 常见问题

### Q: 下载链接打不开？
尝试使用镜像：
- 清华镜像: https://mirrors.tuna.tsinghua.edu.cn/android/repository/platform-tools-latest-windows.zip
- 中科大镜像: https://mirrors.ustc.edu.cn/android/repository/platform-tools-latest-windows.zip

### Q: 解压失败？
确保下载的是 `.zip` 文件，不是 `.exe` 文件。

### Q: adb 命令找不到？
1. 重启命令行窗口
2. 或重启电脑
3. 确保 PATH 环境变量包含 `C:\Users\18268\android-sdk\platform-tools`
