# Tauri Android APK 打包 - 手动安装指南

## 当前状态

- ✅ Java JDK 17 已安装
- ✅ 环境变量已配置
- ❌ Android SDK 需要手动下载（网络限制）

## 解决方案

### 方案一：手动下载并安装（推荐）

#### 1. 下载 Android Studio
在有网络的环境下访问：
- 官网：https://developer.android.com/studio
- 或腾讯软件中心：https://pc.qq.com/detail/0/detail_24680.html

下载后安装 Android Studio，它会自动配置好 Android SDK。

#### 2. 配置环境变量
安装完成后，Tauri 会自动检测到 SDK 路径。

#### 3. 构建 APK
```powershell
cd "d:\word\practise\ai code\Now\phone\now Android\now-android"
npm run tauri android init
npm run tauri android build
```

---

### 方案二：下载命令行工具（轻量级）

#### 1. 下载命令行工具
在有网络的环境下访问：
- https://developer.android.com/studio#command-tools
- 下载 "Command line tools only" (约 150MB)

#### 2. 解压到正确位置
将下载的 zip 解压到：
```
C:\Users\18268\android-sdk\cmdline-tools\latest\
```

#### 3. 安装 SDK 组件
打开 PowerShell，运行：
```powershell
# 设置环境变量（已为你配置好）
$env:ANDROID_HOME = "$env:USERPROFILE\android-sdk"
$env:PATH = "$env:ANDROID_HOME\cmdline-tools\latest\bin;$env:PATH"

# 安装必要组件
sdkmanager "platforms;android-33"
sdkmanager "build-tools;33.0.0"
sdkmanager "ndk;25.2.9519653"
sdkmanager --licenses
```

#### 4. 构建 APK
```powershell
cd "d:\word\practise\ai code\Now\phone\now Android\now-android"
npm run tauri android init
npm run tauri android build
```

---

### 方案三：使用已配置好的环境

如果你在其他机器上有配置好的 Android SDK，可以直接复制过来：

1. 复制整个 SDK 目录到 `C:\Users\18268\android-sdk\`
2. 确保包含以下子目录：
   - `cmdline-tools/`
   - `platforms/` (包含 android-33)
   - `build-tools/` (包含 33.0.0)
   - `ndk/` (包含 25.2.9519653)

- 发布版：`src-tauri/gen/android/app/build/outputs/apk/release/app-release.apk`
